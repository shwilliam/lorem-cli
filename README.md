# Installation

```sh-session
$ npm install -g @shwilliam/lorem-cli
```

# Usage

## `lorem`

```sh-session
USAGE
  $ lorem [PARAGRAPHS]

EXAMPLE
  $ lorem 2
```

## Version

```sh-session
USAGE
  $ lorem (-v|--version|version)
```

## Help

```sh-session
USAGE
  $ lorem --help
```
